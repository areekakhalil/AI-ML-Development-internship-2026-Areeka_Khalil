# 📊 Week 2 – Data Visualization & Exploratory Data Analysis (EDA)

## 📌 Overview
This week's assignment focuses on **Exploratory Data Analysis (EDA)** using the **Titanic dataset**. The goal is to understand the dataset through visualizations before building any machine learning models.

The notebook demonstrates how to create informative charts using **Matplotlib** and **Seaborn**, along with interpreting the insights obtained from each visualization.

---

## 🎯 Objectives

- Understand the importance of Exploratory Data Analysis (EDA)
- Learn to create visualizations using Matplotlib and Seaborn
- Analyze relationships, distributions, and categorical data
- Interpret charts to gain meaningful insights from the dataset

---

## 📂 Dataset

- **Dataset:** `titanic_cleaned.csv`
- The cleaned Titanic dataset from **Week 1**

---

## 📈 Visualizations Created

### 1. Survival Count by Gender
- **Chart:** Seaborn Count Plot
- **Purpose:** Compare survival counts of male and female passengers.

### 2. Age Distribution
- **Chart:** Matplotlib Histogram
- **Purpose:** Visualize the distribution of passenger ages and identify skewness.

### 3. Ticket Fare by Passenger Class
- **Chart:** Seaborn Box Plot
- **Purpose:** Compare fare distributions across passenger classes and detect outliers.

### 4. Correlation Heatmap
- **Chart:** Seaborn Heatmap
- **Purpose:** Display correlations between numerical features in the dataset.

## 📌 Key Insights

- Female passengers had a higher survival rate than male passengers.
- Most passengers were young adults, with the age distribution slightly right-skewed.
- First-class passengers generally paid higher fares and showed more fare variability.
- Passenger class and fare exhibited a strong negative correlation.
